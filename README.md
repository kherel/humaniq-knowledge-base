# For the developers

## Installation

	$ npm i - to install node packages

## Main app

#### production mode

	$ npm run build - to build project.  
	$ npm start - to start node.js server

#### development mode

	$ npm run dev - quick start dev server with ssr   
	$ npm run dev2 - quick start dev server without ssr
